Introducción a la maquetación.

Elementos semánticos

  • HEADER Es el contenido introductorio que contiene los títulos principales, a veces suele tener una galeria a modo de presentación y también la navegación.
    • NAV Es la navegación principal y donde se encuentran solamente los links mas importantes de nuestro sitio.
  • MAIN Es el contenido principal donde colocamos la información relevante para el usuario, la mayoría de los elementos se encontrarán adentro.
    • SECTION Elemento que forma parte del contenido, depende del contexto y solamente funciona dentro de la página, ya que está ligado al contenido de la misma.
    • ARTICLE Elemento independiente al contexto, funciona tanto dentro como fuera de la página y debe ser autoconclusivo.
    • ASIDE Elemento lateral o contenido tangencial que guarda una relación indirecta con sus elementos hermanos.
  • FOOTER Es el contenido de desenlace, podemos encontrar información del autor, datos y/o formularios de contacto, mapas del sitio e incluso links a las redes sociales.

Jerarquia de selectores CSS

  • Tag selector: es aquel que apunta a un tipo de etiqueta, si bien se pueden combinar es el menos especifico y por lo tanto al ser muy general con respecto a los demas, es el que posee menor prioridad ya que no permite distinguir entre las etiquetas que seleccionamos.
    • header{color:darkred}
    • header h1{color:tomato}
  • Class selector: es aquel que apunta a las clases aplicadas sobre los elementos y se diferencia por la implementacion de un punto (.) antes del nombre de la clase. Especificamente permite seleccionar un grupo de etiquetas, incluso pudiento aclarar que tipo de etiqueta es a la que apuntamos.
    • .header{color:darkgreen}
    • .header h1{color:limegreen}
  • ID selector: es aquel que apunta al Id de un elemento en particular, se diferencia por la implementacion de un numeral (#). Es el selector con mayor prioridad porque se considera muy especifico con respecto a los demas, la razon de esto es que no deberian existir mas de una etiqueta con el mismo Id.
    • #header{color:darkblue}
    • #header h1{color:skyblue}
